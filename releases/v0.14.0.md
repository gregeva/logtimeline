# Release v0.14.0

## What's New

- Add `LTL_CONFIG` environment variable for persistent default options (#21)
- Make `-i`, `-e`, `-h`, `-tpa` additive on the command line — multiple uses combine with OR (#21)
- Add persistent index file (`ltl-index.csv`) that captures per-file metadata after each run (#46)
- Add AND operator (`&`) for `-i`, `-e`, `-h` filter patterns — `A&B` matches lines containing both A and B; `&&` for literal `&` (#117)
- Add `-ru`/`--rate-unit` option to configure rate normalization unit: `s` (per-second), `m` (per-minute, default), `h` (per-hour), `d` (per-day) (#122)
- Documentation overhaul: restructure README as landing page, add `docs/purpose.md` design philosophy, create `docs/usage.md` full options reference with sectional explanations and examples, rewrite `print_help()` with ANSI-formatted headings and dynamic word-wrapping (#118)
