# Release v0.12.1

## Bug Fixes

- Fix uninitialised `$scaled_trend_value` warning when hiding occurrences, duration, and bytes columns with `-ho -hd -hb` (#86)
- Fix heatmap/histogram colors not matching bar graph metric column colors — resolve all colors from `@column_layout` instead of hardcoding at init (#83)
- Document UDM aggregation functions, transforms, units, and regex pattern syntax in `--help` and README (#84)
- Fix macOS build failure by using `perl -S pp` to find the correct PAR::Packer (#85)
- Fix highlighted message count mismatch when thread names contain nested brackets, e.g. C3P0 connection pool managers (#62)
