# Release v0.12.0

## What's New

- Unified column layout engine replaces 5 overlapping data structures with single `@column_layout` array-of-hashes (#33)
- Regression test infrastructure with 15 reference output tests (#33)
- Hidden CLI options for debugging: `--terminal-width`, `--debug-layout`, `--validate-layout` (#33)
- Automatic Sessions column shows unique concurrent sessions per time bucket when session IDs are found in logs; suppress with `--hide-session|-hs` (#26)
- Thread pool and session column values now use `format_number()` for readable display at narrow widths (#26)
- Remove legacy `$durations_graph_width` variable; heatmap footer now uses column layout for positioning (#26)
- Add `--hide-*` options for all standard columns: `-hl`, `-ho`, `-hd`, `-hb`, `-hc`, `-hst` (#75)

## Bug Fixes

- Fix CSV output (`-o`) to include raw session/threadpool counts instead of display-formatted values (#74)
- Fix line wrapping when using `-tpa` with additional proportional columns (#27)
- Fix UDM rendering errors and crashes when using many `-udm` columns (#69)
