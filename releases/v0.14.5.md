# Release v0.14.5

## What's New

- Re-running ltl against files it has already analyzed is now faster: prior runs' bound information is read from `ltl-index.csv` at startup, skipping the initial bound-discovery scan (#179)
- Histogram bottom axis (`-hg`) now shows tick marks at each percentile position so legend values line up visually with the distribution (#185)
- New `--percentile-precision` / `-pp` (tier 1–9) and `--percentile-buckets-per-decade` / `-pbpd` flags to tune percentile accuracy; `--exact-percentiles` / `-ep` opt-out for the previous sort-based computation (#189)
- Heatmap (`-hm`) and histogram (`-hg`) use ~60% less memory and run noticeably faster on large files (~120× on heatmap calculation on a 148 MB Tomcat log). Output is visually identical to previous releases. Use `--exact-percentiles` to fall back to the previous behavior (#34)
