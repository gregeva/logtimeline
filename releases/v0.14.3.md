# Release v0.14.3

## What's New

- Default Dice similarity threshold for `-g` and `--final-threshold` changed from 80% to 85%
- Added "Consolidating similar messages..." progress status display during fuzzy consolidation stage
- Benchmark script (`run-benchmark.sh`) now merges results when run incrementally — running `full` then `xl` separately no longer overwrites previous results
- Standardized NYTProf profiling workflow with observability framework: `run-profile.sh`, `extract-profile.pl` (with declarative `--checks-file` cross-validation), profile-ready contract, pre/post-profiling checklists, and diagnostic pattern index (#138)
