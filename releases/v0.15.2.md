# Release v0.15.2

## Bug Fixes

- Single-dash long forms of the informational options now work again: `-version`, `-help`, and `-explain` behave identically to their `--` forms, instead of failing with a misleading "unable to open any files" error.
- `-help` and `-explain` no longer print the banner twice.
- `-?` is now documented as the short form of `--help` in the help screen and the usage reference.
- `--help`, `--explain`, and `--version` output now reflows to the width set by `--terminal-width`.
- `--help <unknown-topic>` now exits with an error naming the topic, consistent with `--explain`, instead of silently showing the full help screen.
- Removed help-screen and usage-documentation text that described internal option-handling behavior rather than user-facing usage.

## Performance

CLI-only bug-fix release with no changes to the log-processing path; no benchmark comparison is included.
