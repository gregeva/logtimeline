## What's New

- **GitHub Actions CI/CD Pipeline**: Automated cross-platform builds triggered on version tags
  - Builds 4 platform binaries automatically
  - Attaches binaries to GitHub Releases
  - Manual trigger support for testing (`workflow_dispatch`)

- **Release Notes Requirement**: All releases now require release notes in `releases/v{version}.md`

## Build Script Improvements

- `macos-package.sh`: Added architecture parameter, build verification, dynamic module includes for PAR::Packer
- `ubuntu-package.sh`: CI-compatible (removed `-it` flag), architecture parameter support, build verification

## Output Binaries

| Platform | Binary |
|----------|--------|
| macOS (ARM64) | `ltl_static-binary_macos-arm64` |
| Linux (amd64) | `ltl_static-binary_ubuntu-amd64` |
| Linux (arm64) | `ltl_static-binary_ubuntu-arm64` |
| Windows (amd64) | `ltl_static-binary_windows-amd64.exe` |
