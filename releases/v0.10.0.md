## What's New

- **Histogram Charts**: New `-hg` / `--histogram` option displays ASCII histogram charts visualizing the distribution of duration, bytes, and count metrics across all log entries. Histograms appear after the time-series bar graph and provide at-a-glance distribution visualization to identify common value ranges, outliers, and patterns (normal, bimodal, long-tail).

### Histogram Features

- **Multiple metrics**: Display histograms for duration, bytes, count, or any combination side-by-side
- **HdrHistogram-style bucketing**: Uses industry-standard buckets-per-decade model for statistically meaningful logarithmic binning
- **Dynamic scaling**: Histogram height adapts to terminal size (4-15 rows), width uses 95% of terminal by default
- **Dual Y-axes**: Left axis shows absolute count, right axis shows percentage
- **Sub-character resolution**: 8-level Unicode block characters (▁▂▃▄▅▆▇█) for smooth gradients
- **Percentile legend**: Priority-based selection shows P50, P99, P99.9, P95, etc. based on available width
- **Color support**: Matches bar graph column colors (yellow/green/cyan), with optional gradient mode
- **Light background support**: Works with `-lbg` flag for light terminals

### New Command Line Options

| Option | Description |
|--------|-------------|
| `-hg [metric]` | Enable histogram (duration, bytes, count, or comma-separated) |
| `-hgw <N>` | Set histogram width as percentage of terminal (default: 95) |
| `-hgh <N>` | Set histogram height in rows (default: 8) |
| `-hgbpd <N>` | Buckets per decade for precision control (default: 8) |
| `-hgb <N>` | Override with explicit bucket count |
