# Release v0.12.5

## Bug Fixes

- Fix UDM CSV auto-detection misidentifying log files as CSV; detection now uses lazy two-line validation (#107)
- Fix number formatting regex mangling values like `64.09 k` into `649 k` in `format_number`, `format_bytes`, and `format_time` (#108)

## Build

- Harden Windows build against GitHub API rate limiting by authenticating with `GITHUB_TOKEN` and adding retry logic (#106)
